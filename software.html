PhotoStorm v0.1 - THIS IS AN OLD VERSION

PhotoStorm Desktop for Linux is divided in four packages + the source:
  • Install Script
  • Uninstall Script
  • Binaries v0.1
  • Data Files v0.1
  • Source v0.1

The first two are the scripts for installing/uninstalling photostorm in /usr/local. If you want to install photostorm manually, please read the instructions below. This package should be updated from time to time.

The binaries package contains the core files (executables) of PhotoStorm. They have been compiled for Linux, under Fedora Core 1, GCC 3.2

The data package contains the files needed for photostorm to run properly. It might start without them, but it won't work properly. Usualy this package is downloaded only once and does not need updates.

The source is not very stable yet and we don't recomand you to use it as it will be difficult to compile. Use it just in case the binaries do not work.

I. Installing PhotoStorm

Read the following instruction in order to install PhotoStorm
  • Download the first four packages (scripts + binaries + data)
  • Login as root
  • Make sure that the scripts have the permission to run (chmod a+x install_photostorm.sh uninstall_photostorm.sh)
  • Run install_photostorm - this will install photostorm in /usr/local/bin - binaries and /usr/local/lib/photostorm
  • If the script runs correctly, you should now be able to start photostorm. Please read the section Running PhotoStorm
II. Installing PhotoStorm Manually

Read the following instruction in order to install PhotoStorm manually, without the install script
  • Download the two packages (binaries + data)
  • Login as root
  • Untar the binaries tar xvfz photostorm-bin-0.1.tgz -C "location to place binaries" 
    eg:
	tar xvfz photostorm-bin-0.1.tgz -C /usr/bin
    This will install photostorm binaries in /usr/bin
  • Untar the binaries tar xvfz photostorm-bin-0.1.tgz -C "location to place binaries" 
    eg:
	tar xvfz photostorm-data-0.1.tgz -C /usr/lib
    This will install photostorm data files in /usr/lib in a folder photostorm.
III. Running PhotoStorm

Read the following instruction in order to run PhotoStorm
  • You can start the gnome panel applet from Add to Panel -> Multimedia -> PhotoStorm This applet controls the updating of the wallpaper and its functions. You should run it.
  • You can start PhotoStorm by simply typing photostorm in the comand line.
IV. Building the source code

Read the following instruction in order to build PhotoStorm from its source code
  • Untar photostorm-src-0.1.tgz (tar xvfz photostorm-src-0.1.tgz)
  • You should have now a folder called photostorm-0.1
  • Change the folder to photostorm-0.1
  • Type make all - this will build photostorm and ps_wallpaper
  • Change the directory to applet
  • Type make applet - this will build ps_applet
  • Now you have the three binaries, you have to copy them in a binaries folder (/usr/bin, /usr.local.bin, or anywhere else in the search path
  • Install the photostorm data pakcage (see part I)
  • PhotoStorm yould work now
If you have any problems with building the source, please mail at source@photostorm.net
V. Troubleshooting

If you havce any problems with PhotoStorm, please try one of the following
  • If you have any problems with using photostorm, please email at contact@photostorm.net
  • If you have any bugs, please email at bugs@photostorm.net

PhotoStorm 0.1.1 - FIXED GTK+ WARNINGS FOR Windows 98/Me
PhotoStorm Desktop for MicrosoftTM WindowsTM is available in two packages:
  • PhotoStorm Desktop Win32-0.1.1 with GTK 2.4.3-Rev-a Libraries (~6.5 MB)
  • PhotoStorm Desktop Win32-0.1.1 without GTK Libraries (~2.7 MB)
You may also download the GTK 2.4.3-Rev-a Runtime installer.
IMPORTANT!!!

PhotoStorm Desktop for MicrosoftTM WindowsTM requires the GTK Runtime Libraries 2.4 or higher. You may install them by yourself, but the registry settings must be the same as the ones made by the GTK Libraries Installer.

If you are not sure about what to download, we recommand you the first package, the one with the GTK Libraries included.

Please note that PhotoStorm Desktop will install the WIMP GTK Theme Engine.

PhotoStorm Desktop is available only for MicrosoftTM WindowsTM and Linux. For other systems, you can use the online wallpaper feature. Just choose a picture that you like and click Make Wallpaper or the image .
